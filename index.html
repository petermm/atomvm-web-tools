AtomVM ESP32 Web Flasher

Easy install of the AtomVM runtime on ESP32 devices in the browser:

  • Install & update AtomVM runtime
  • Access logs and send terminal commands

Install AtomVM runtime

This will install AtomVM v0.6.1. To get started, connect an ESP32 device to your computer and hit the install button:

The Web Flasher is not available because your browser does not support Web Serial. Open this page in Google Chrome or Microsoft Edge instead (but not on your iOS device).

Once completed you are then ready to install/flash and run your own code using rebar3 esp32_flash or for elixir mix atomvm.esp32.flash in your project.

This is merely a fork adopted for AtomVM see upstream esp-web-tools.
